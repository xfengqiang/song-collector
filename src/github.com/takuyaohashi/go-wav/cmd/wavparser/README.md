wavparser - wav header parser
===
[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

# About
wavparser parse wav file header

# Install

```bash
$ go get github.com/takuyaohashi/go-wav/cmd/wavparser
```

# Usage

```bash
$ wavparser wavefile 
```
